# Adaptive Exits (Richtlinie)

- Erlaubte Exit-Logiken u. a.: Scale-out, Runner, Time-Stop, Liquidity-Ziel, Event-Exit, Trend-Hold, Mean-Reversion-Unwind.
- Kein globales starres Take-Profit-Muster; konkrete Trigger kommen aus Playbook + Exit-Engine + Risk-Governor.
- Post-Trade-Reviews sollen Planteil vs. tatsächliche Exit-Familie vergleichen, ohne neue Parameter zu setzen.
