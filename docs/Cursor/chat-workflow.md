# Cursor-Chat: Ablauf mit Prompts, Video und Screenshots

So arbeiten wir reproduzierbar im Repo:

## 1. Im Repository ablegen

| Was                                         | Wo                                                              |
| ------------------------------------------- | --------------------------------------------------------------- |
| Prompt-Sammlung / Checklisten (MD)          | z. B. `docs/Cursor/*.md` (z. B. `25-prompts-production-fix.md`) |
| Video                                       | `docs/Cursor/assets/videos/`                                    |
| Screenshots                                 | `docs/Cursor/assets/screenshots/`                               |
| Optional: stichwortartige Notizen zum Video | `docs/Cursor/video-review-notes.md`                             |
| Methodik: Video inhaltlich voll erfassen    | `docs/Cursor/video-analysis-guide.md`                           |

Dateinamen moeglichst sprechend: `thema-kurzbeschreibung.png`, `thema-session.mp4`.

## 2. Im Cursor-Chat

1. **Einen konkreten Prompt** aus der passenden MD-Datei kopieren (oder den Abschnitt zitieren).
2. **Screenshots** aus `docs/Cursor/assets/screenshots/` an den Chat anhaengen, die genau zu diesem Prompt / dieser UI-Stelle passen.
3. **Optional:** einen kurzen Ausschnitt aus `docs/Cursor/video-review-notes.md` beilegen (Timestamp, Beobachtung, Erwartung vs. Ist).

Damit bleiben Kontext, Evidenz und Arbeitsauftrag zusammen und sind im Repo nachvollziehbar.
