# German UI Glossary

Verbindliches Glossar fuer die sichtbare Main-Console-Oberflaeche.

| Englisch / Altbegriff | Verbindlicher deutscher UI-Begriff |
| --- | --- |
| Main Console | Hauptkonsole |
| Dashboard | Konsole oder Übersicht (kontextabhängig) |
| Operator | Betreiber |
| Admin | Administrator oder Owner |
| Live Trading | Echtgeldmodus oder Live-Modus |
| Paper Trading | Papiermodus |
| Shadow Trading | Schattenmodus |
| Readiness | Bereitschaft |
| Health | Systemzustand |
| Risk | Risiko |
| Signal | Signal |
| Strategy | Strategie |
| Asset | Asset (konsistent beibehalten) |
| Kill Switch | Not-Stopp |
| Safety Latch | Sicherheits-Sperre |
| Reconcile | Abgleich |
| Drawdown | Drawdown |
| No Trade | Kein Handel |
| Quarantine | Quarantäne |
| Live Blocker | Live-Blocker |

## Technische Ausnahmen

Erlaubte englische Begriffe, wenn sie nicht als Produktlabel, sondern als
technischer Fachbezug erscheinen:

- API-/ENV-/Header-Namen (`API_GATEWAY_URL`, `DASHBOARD_GATEWAY_AUTHORIZATION`)
- Protokoll- oder Standardbegriffe (`SSE`, `JSON`, `WebSocket`)
- interne Service-/Tabellennamen und IDs
- dokumentierte Feldnamen in Debug-/Forensik-Ansichten
