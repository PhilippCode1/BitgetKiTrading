# schemas

- `drawing.schema.json` — Drawing-Records (Prompt 12, `drawing-engine`).
- `signal_explanation.schema.json` — lange JSON-Erklaerung (Prompt 14, signal-engine; später LLM structured output).
- `news_summary.schema.json` — strukturierte News-Zusammenfassung (Prompt 16/17, llm-orchestrator).
